타사이트 반품·교환 메뉴얼
🚫 절대 잊지 말 것
1셀메이트 CS 기재 무조건 — 누가 봐도 알게 상세히
2보류 설정 필수 — 안 하면 자동 환불됨
3불량은 사진 필수 — 없으면 문자 발송
4취소 체크 완료까지 처리 — 누락 시 사이트 주문 살아있어 고객 지연 컴플레인 (네이버페이 품절은 재고 0으로 수동 변경)
5환불 계좌 별도 기재 — 본문에 묻히지 않게 셀메이트에 따로 한 줄로
6"고객 직접 발송" 체크 확인 — 자동 수거 채널이라도 체크돼 있으면 CS 메모 (예: "고객 직접 발송 / 배송비 변동됨") · 에이블리 제외
📋 공통 처리 흐름
① 요청 탭 확인 → ② 사유·사진 확인 → ③ 셀메이트 기재
→ ④ 사진 첨부 확인 → ⑤ 보류 설정 → ⑥ 반품/교환 접수
네이버페이 자동 수거
NAVER PAY · 수거는 자동, 사유 확인 + 기재만
STEP 1
반품 관리 탭 들어가서 요청 건 확인
반품 사유, 반입 수량, 메모 내용 다 확인한다. 수거는 자동이라 따로 신청 안 해도 되지만, 반품 사유는 반드시 확인.
⚠️ 자주 놓치는 부분 — 부분 반품 체크
반입 수량은 부분 반품이 안 됨. 만약 2개 주문에 반입 수량이 2개로 떠 있는데, 고객이 메모에 "하나만 반품한다"고 쓴 경우가 있음. 그런 건 메모 꼭 확인하고 셀메이트에 기재.
STEP 2
단순 변심 vs 오배송/지연/파손 구분
단순 변심: 수량 외엔 크게 볼 거 없음. 셀메이트 기재만 하고 넘어가면 됨.
오배송/지연/파손/불량: 특이사항 없어도 무조건 셀메이트 기재 + 보류 설정.
STEP 3
파손/불량/오배송 건은 보류 설정 + 셀메이트 기재
고객이 남긴 내용과 첨부 사진이 있으면 같이 확인. 누가 봐도 내용 확인할 수 있도록 상세하게 셀메이트에 기재.
💡 셀메이트 기재 예시
"파손 - 원피스 옆선 부분 실밥 풀림, 사진 첨부 확인됨"
"오배송 - L 주문건인데 M 수령, 사진 확인됨"
"단순변심 - 사이즈 안 맞음, 특이사항 없음"
STEP 4
교환 관리 탭 확인 → 초기 보류 설정 + 재고 확인
단순 변심 사이즈 교환이라도 무조건 초기 보류 설정 1회는 해줘야 함.
🚫 보류 안 하면 생기는 일
수거 완료되면 자동으로 주문이 생성됨. 물건 안 들어왔어도 시스템상으론 처리되는 거라 나중에 꼬임. 불량이든 변심이든 접수되는 거 다 보류 한 번씩.
⚠️ 교환 접수 전 재고 현황 확인 필수
시즌 지나면 품절 건 많음. 교환 접수 전 무조건 재고 확인:
재고 상황처리 방법
재고 1-2개만 남음반품하고 재구매 유도 (유도리 있게 안내)
완전 품절교환 불가 안내
정상 재고교환 진행
❓ 보류해야 되는지 어떻게 알아요?
요청 탭과 수거 탭 두 군데에서 보류가 안 돼 있으면 무조건 보류. 수거 완료 후 보류 해제돼 있는 건 은희님이 처리한 거니까 상관없음.
지그재그 자동 수거
ZIGZAG · 3개 탭(취소/반품/교환) 다 확인 필수
STEP 1
취소 요청 / 반품 요청 / 교환 요청 탭 전부 확인
메인 화면에서 건수 떠 있는 거 하나씩 다 눌러서 들어가야 함. 없으면 상관없는데 떠 있으면 다 셀메이트로 긁어옴.
⚠️ 자주 놓치는 부분
예전에는 취소 요청만 보고 나머지는 고객 문의 들어오면 처리했는데, 이제는 3개 탭(취소/반품/교환) 다 봐야 함.
STEP 2
배송 일자 + 반품 접수일 확인 → 접수 또는 거부
특이사항 없으면 그냥 접수해주면 됨. 접수 안 해주면 택배사가 수거를 안 감.
⚠️ 거부 처리 기준 — 배송일 경과
우리 내부 기준: 배송 완료일 포함해서 7일
타 사이트 기준: 배송 완료일 다음 날부터 7일
→ 우리 기준으로 경과하면 거부 처리하고 밑에 상세 사유 기재. 그러면 다시 백 들어옴.
STEP 3
교환 요청 — 재고 확인 + 배송비 결제 방식 확인
교환 요청 클릭하면 밑에 교환 처리 정보 뜸. 거기서 비용 결제 방식 확인:
비용 결제 방식셀메이트 기재
바로 결제 (고객이 접수하면서 결제 완료)"사이즈 교환, 즉시 결제 건"
송금 (나중에 입금 받아야 함)"사이즈 교환, 고객 송금 예정"
⚠️ 교환 접수 전 재고 현황 확인 필수
시즌 지나면 품절 건 많음. 교환 접수 전 무조건 재고 확인:
재고 상황처리 방법
재고 1-2개만 남음반품하고 재구매 유도 (유도리 있게 안내)
완전 품절교환 불가 안내
정상 재고교환 진행
💡 왜 이렇게 적어야 하나
나중에 또 들어가서 확인하면 중복 작업됨. 처음부터 결제 방식까지 적어놔야 한 번에 끝남.
STEP 4
교환 요청이 시스템엔 없는데 CS로 문의만 들어온 경우
고객이 시스템 접수 안 하고 채널톡/문의로만 교환 요청한 경우가 있음. 이때는 셀메이트에:
"고객이 교환 요청 CS로 왔는데 시스템에는 안 함. 요청 안내 완료."
고객한테는 이렇게 안내:
"교환 요청 및 수거 접수, 배송비 결제까지 진행해 주시면 됩니다."
에이블리 수동 수거
ABLY · 수거 접수 직접 처리 필요
⚠️ 에이블리 특징 — 자동 수거 안 됨
네이버페이/지그재그와 다르게 수거를 수동으로 접수해줘야 함. 우리가 접수 안 누르면 택배사 안 옴.
STEP 1
반품/교환 요청 건 확인 후 셀메이트 기재
"1년 조회합니다" 같은 팝업 뜨면 확인 누르고 진행. 전체 다 뜨는데 그 중 요청 접수 건만 골라서 셀메이트로 긁어옴. 발송 일자 확인.
STEP 2
반품 접수 → 자동 수거 접수까지 이어서 처리
체크해서 반품 접수 → 이어서 요청 접수 누르면 "수거 접수해 주세요" 뜸 → 요청 누르면 검수 대기로 넘어감.
STEP 3
교환 탭도 동일하게 처리 — 재고 확인 필수
에이블리에 교환 탭이 얼마 전 새로 생김. 반품이랑 처리 방식 똑같음. 떠 있으면 체크 → 교환 접수 → 셀메이트 기재 → 수거 접수.
⚠️ 교환 접수 전 재고 현황 확인 필수
시즌 지나면 품절 건 많음. 교환 접수 전 무조건 재고 확인:
재고 상황처리 방법
재고 1-2개만 남음반품하고 재구매 유도 (유도리 있게 안내)
완전 품절교환 불가 안내
정상 재고교환 진행
💡 불량 접수 건 셀메이트 기재 예시
"교환 요청 - 판매자 귀책 / 기름 얼룩 불량 접수 / 위치: 앞면 오른쪽 하단 / 사진 첨부 확인됨"
비플로우 자동/수동 혼합
B-FLOW · 채널별로 처리 방식이 다름 (가장 복잡)
🚫 비플로우는 제일 복잡함
비플로우 안에 판매 채널이 여러 개(지마켓, 옥션, 11번가, 할프클럽 등)라서 채널마다 처리 방식이 조금씩 다름. 헷갈리면 무조건 1:1 문의로 비플로우에 물어보기.
STEP 1
⭐ 회수 방법 확인 — 자동인지 수동인지
반품 요청 들어왔을 때 반품 번호 클릭해서 안에 들어가서 회수 방법 확인.
"지정 택배로 자동 회수가 진행됩니다" → 우리가 수거 안 해도 됨
"고객이 회수 요청을 했습니다. 진행해 주세요" → 에이블리처럼 수거 접수 필요
[참고] 비플로우 반품 페이지 — 회수수거방법 항목 (붉은 네모)
STEP 2
⭐ 빠른 환불 채널 — 지마켓·옥션 주의
지마켓, 옥션 같은 데는 "환불 배송비 처리 조건: 빠른 환불"이라고 떠 있음. 이건 자동 회수 건이라:
🚫 빠른 환불 채널의 특징
택배사가 수거 송장만 띄워도 물건 안 들어왔는데 환불이 자동으로 나가버림.
→ 요청 탭에 안 뜨고 바로 완료 탭으로 넘어가 있음.
→ 빠른 환불 건은 별도 보류 처리가 안 됨 (STEP 3 보상/보류요청 불가)
⚠️ 처리 방법 — 셀메이트에 이력 남기기
보류가 안 되니까 셀메이트에 이력만 남겨주면 됨. 반입 시 은희님이 이상 유무를 확인해줌.
💡 셀메이트 이력 기재 예시
"반품 / 빠른 환불건 이상유무 확인요청"

처리 흐름
① 빠른 환불건 확인 → ② 셀메이트에 이력 기재 → ③ 반입 시 은희님이 확인 → ④ 이상 있으면 비플로우 1:1 문의로 보상신청
💡 비플로우 안내 내용 (참고)
"빠른 배송의 경우 반품송장의 흐름이 확인될 경우 즉시 환불하는 지마켓/옥션 시스템. 배송비가 필요할 경우 판매자가 보상신청을 하는 채널이 별도로 존재. 불량이 아니라고 판단되면 상세 내용 문의하기 남겨주시면 판매채널 보상요청 진행."
STEP 3
⭐ 불량 접수건 환불 보류 방법 — 보상/보류요청
비플로우는 "보상/보류요청" 기능으로 환불을 보류함. (W컨셉의 회수 관리 메모와 다름!)
⚠️ 판매처별로 접수 가능 여부 다름
비플로우 안 채널마다 다름 — 일단 접수 시도해보고, 안 되는 채널은 비플로우 1:1 문의로 확인.
처리 순서
① 반품 목록에서 해당 건 체크 → "보상/보류 요청" 버튼 클릭
[참고] 반품 목록 상단 — "보상/보류 요청" 버튼 (붉은 네모)
② 보상/보류요청 사유 선택 (보통 "기타" 선택)
③ 보상/보류요청 상세사유 기재 (예: "기간경과 반품 불가건 / 철회 요청")
④ 관리자 메모 작성 (예: "채널 문의글로 클레임 접수완료")
[참고] 보상/보류 요청 작성 폼 — 사유/상세사유/관리자메모 (붉은 네모 부분 입력)
💡 접수 후 답변 확인
접수 완료 후 하단 관리자 메모 (비플로우 답변)에서 비플로우 처리 결과 확인 가능. 비플로우 측에서 답변 달리면 알람으로 옴.
STEP 4
불량 접수인데 사진이 안 보일 때 → 비플로우에 문의 남기기
고객이 사진을 첨부했는데 우리한테 안 보일 때가 있음. 이런 경우 비플로우 1:1 문의로 사진 확인 요청.
문의 작성 방법
1. 1:1 문의 → 구분: 판매관리 / 반품관리·환불보류 선택
2. 상품 주문번호 입력 (브릿지 주문번호 아님 — 비플로우 안에서 보이는 상품 주문번호)
3. 제목: "불량 첨부 사진 확인요청"
[참고] 비플로우 1:1 문의 작성 화면 — 구분/상품주문번호/제목/본문
STEP 5
셀메이트 기재 + 사진 확인 상태 표시
불량 내용 똑같이 남기되, 사진 상태를 표시:
상황셀메이트 기재
사진 첨부 안 됨 → 비플로우 문의 중"불량 - [내용] / 사진 확인 문의 중"
사진 확인됨"불량 - [내용] / 사진 OK"
고객한테 직접 채널톡으로 사진 요청"불량 - [내용] / 채널톡 사진 요청 발송"
STEP 6
수거 주소지 확인 — 배송 주소와 다른지
고객이 수거 주소를 변경해서 접수하는 경우가 있음. 수거 주소지가 배송 주소지와 다르면 접수할 때 변경 처리.
⚠️ 변경 건이라고 따로 뜨지 않음
시스템에서 "변경 건"이라고 알려주지 않음. 직접 두 주소를 비교해서 확인해야 함.
STEP 7
배송비 정산 — 마이너스/0/금액 표시 의미
반품 페이지 안의 배송비 정산 항목 확인:
표시의미 / 처리
6,000원 등 정상 금액자동 차감됨 → 별도 처리 없음
0원 (불량 접수 건)들어오면 확인 후 입금 안내 (수정님 후처리 X)
마이너스고객이 입금해야 함 → 셀메이트 CS 메모에 남기기
STEP 8
수거 송장번호 처리 (수동 수거 건)
비플로우는 수거 송장번호를 입력해야 함. 다음 날 수거 시스템에 송장번호가 자동으로 뜸.
우리가 수동으로 수거 접수한 건은 다음 날 수나 언니가 확인 후 처리해줌. 그래서 수거 접수만 정확히 해놓으면 됨.
💡 송장번호 처리 흐름
① 오늘: 수동으로 수거 접수만 완료
② 다음 날: 수거 시스템에 송장번호 자동 생성
③ 수나 언니가 확인 후 송장번호 처리
⚠️ 우리가 따로 송장번호 안 넣어도 됨
수거 접수가 정확히 됐는지만 확인하고 넘기면 됨. 송장번호 입력은 수나 언니 담당.
W컨셉(더블유컨셉) 자동 수거
W CONCEPT · 자동 수거지만 회수 관리 메모 필수
STEP 1
자동 수거 — 클레임 사유만 확인
W컨셉은 자동 수거라서 특별히 볼 건 없음. 클레임 사유 확인 + 셀메이트 기재만 진행.
STEP 2
불량 접수 — 첨부 파일 확인
클레임 사유 옆 첨부 파일 클릭하면 사진 보임. 첨부 안 한 고객은 "없음"으로 표시.
STEP 3
애매한 불량 → 고객에게 안내
누가 봐도 불량 아닌 건이면 셀메이트에 기재하면서 고객한테 안내:
"불량으로 보기 어렵습니다. 반입되면 정확하게 확인 가능합니다."
STEP 4
⭐ 회수 관리 메모 무조건 남기기 (자동 환불 방지)
W컨셉도 비플로우처럼 수거 완료되고 일정 기간 지나면 자동 환불됨. 회수 관리 처리 보류 메모를 남겨놔야 자동으로 처리 보류됨.
🚫 메모 안 남기면
클레임 사유 떠 있는데 메모 없으면 → 시간 지나면 환불 자동 처리됨 → 우리가 확인할 시간 없어짐
